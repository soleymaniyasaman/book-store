About Book Store

Application Overview

Book Store is a modern Angular application designed to help you manage your book collection. Built with the latest Angular features and best practices, it demonstrates real-time updates, responsive design, and modern state management techniques.

Key Features

  • Real-time Updates: Book data changes are propagated throughout the application in real-time
  • Responsive Design: The application works seamlessly across desktop, tablet, and mobile devices
  • Theme Support: Toggle between light and dark themes based on your preference
  • Advanced Search: Quickly find books by title, author, or description
  • Event Sourcing: All changes to the book list are tracked as events

Technical Implementation

This application has been built using:

  • Angular 18 (Latest Stable Version)
  • RxJS for reactive state management
  • Angular Material for UI components
  • Cypress for end-to-end testing
  • SCSS for advanced styling with theming support
